# **App Name**: WebHookPulse

## Core Features:

- Custom Webhook Endpoint: Generate a unique URL to receive and parse incoming POST payloads in real-time.
- AI Payload Interpreter: A smart tool that uses generative AI to analyze incoming JSON data and extract specific codes or patterns automatically.
- Live Activity Stream: A dashboard interface that displays received codes and payload details instantly using real-time updates.
- Code Persistence: Save all extracted codes and original payloads to a persistent database for future reference.
- JSON Payload Inspector: A dedicated viewer for exploring the full raw data received from external services.

## Style Guidelines:

- Primary color: Electric Blue (#3380FF) for a technical, data-focused energy that feels modern and precise.
- Background: Deep Charcoal (#121417), a dark scheme chosen to mimic developer tools and reduce eye strain during long monitoring sessions.
- Accent: Indigo Violet (#7B61FF) to provide a sophisticated contrast for action buttons and high-priority code status.
- Headline font: 'Space Grotesk' (sans-serif) for its technical and scientific aesthetic. Body font: 'Inter' (sans-serif) for high legibility in dense data tables.
- Code font: 'Source Code Pro' (monospace) for displaying all webhook payloads and extracted snippets.
- A dashboard-style grid featuring a sidebar for payload history and a large central viewport for real-time inspection.
- Pulse effects for new incoming requests and subtle smooth-scrolling as the data feed updates.
